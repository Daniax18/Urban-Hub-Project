# Utilisation de l'IA

Ce document indique l'utilisation de l'IA dans la préparation des documents de l'épreuve.

> L'IA n'a pas été utilisée pour remplacer le travail technique, mais pour améliorer la qualité de présentation, la clarté des documents et la cohérence des formulations.

## 1. Outil IA utilisé

L'outil IA utilisé est :

```text
ChatGPT
```

## 2. Parties concernées de l'épreuve

L'IA a été utilisée comme aide à la rédaction et à la mise en forme.

Les parties concernées sont :

- correction du vocabulaire ;
- correction des fautes d'orthographe ;
- correction du format Markdown ;
- proposition de phrases plus cohérentes, simples et claires ;
- aide à la formulation de la gravité des vulnérabilités Snyk.

## 3. Prompts utilisés

Exemples de prompts utilisés :

```text
Pour les vulnérabilités Snyk suivantes, quelle peut être la gravité de chacune ?
```

```text
Corrige les fautes d'orthographe et de grammaire dans ce document Markdown, sans changer le sens technique.
```

```text
Reformule cette partie avec des phrases claires et cohérentes pour un rapport d'épreuve.
```
